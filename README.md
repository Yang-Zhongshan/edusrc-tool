# EDUSRC 漏洞情况统计脚本

## 一、安装

```bash
git clone https://github.com/Yang-Zhongshan/edusrc-tool
cd edusrc-tool
pip3 install -r requirements.txt
python3 main.py -u 杨众山
```

## 二、使用说明

```
用法：python3 main.py [-new] [-u] [-s] [-help] target
选项：
    -new   user/school   更新用户/学校信息
    -u     username      查询用户信息
    -s     schoolname    查询学校信息
    -help                帮助信息
```
